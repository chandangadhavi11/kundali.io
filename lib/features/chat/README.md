# Chat Feature

## Overview
AI-powered astrology chat assistant for personalized guidance and kundli interpretations.

## Screen
- `screens/ai_chat_screen.dart` - Main chat interface

## Widgets
| Widget | Description |
|--------|-------------|
| `ai_assistant_view.dart` | AI chat conversation view |
| `astrologers_marketplace_view.dart` | Professional astrologer listings |
| `chat_history_view.dart` | Past conversations list |

## Features
- AI-powered astrology assistant
- Context-aware responses
- Kundli-based personalized advice
- Quick question suggestions
- Chat history persistence
- Voice input support (planned)
- Image sharing for palm reading (planned)

## Chat Modes
- General astrology questions
- Kundli interpretation
- Daily guidance
- Relationship advice
- Career guidance
