# Home Feature

## Overview
The main dashboard screen featuring a premium cosmic dark theme with elegant glassmorphism effects, beautiful animations, and seamless navigation to all app features.

## Design Philosophy
- **Premium Cosmic Aesthetic**: Deep space theme with animated star fields and gradient backgrounds
- **Glassmorphism**: Frosted glass effects with subtle borders and shadows
- **Elegant Typography**: Clean, modern fonts with proper hierarchy
- **Smooth Animations**: Staggered entry animations and floating effects
- **Intuitive UX**: Clear visual hierarchy and easy navigation

## Screen
- `screens/home_screen.dart` - Main premium home dashboard

## Sections
1. **Premium Header**
   - Brand logo with golden gradient
   - Personalized greeting with time-based emoji
   - Live time display
   - Notification and profile buttons

2. **Cosmic Energy Card**
   - User's zodiac sign with animated symbol
   - Daily horoscope preview
   - Mood, lucky color, and lucky number metrics

3. **Quick Actions Grid**
   - Primary action cards (Kundali, Match Making)
   - Secondary actions (Horoscope, Panchang, AI Guru, Learn)

4. **Panchang Card**
   - Today's Hindu calendar details
   - Nakshatra, Yoga, Karana information
   - Sunrise and Sunset times

5. **AI Astrologer Banner**
   - Premium CTA for AI consultation

## Features
- Animated cosmic background with stars
- Parallax scrolling effects
- Pull-to-refresh support
- Staggered animations on load
- Haptic feedback on interactions
- Guest mode with sign-in prompt
- Responsive layouts
