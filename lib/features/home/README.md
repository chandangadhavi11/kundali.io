# Home Feature

## Overview
The main dashboard screen showing quick access to all features, daily updates, and personalized content.

## Screen
- `screens/home_screen.dart` - Main home dashboard

## Sections
- User greeting with zodiac info
- Quick action cards (Generate Kundli, Compatibility, etc.)
- Today's Panchang summary
- Daily horoscope preview
- Featured content carousel
- Recent kundalis list

## Features
- Personalized greetings based on time of day
- Real-time panchang data display
- Quick navigation to all features
- Animated entry transitions
- Pull-to-refresh support
- Responsive grid layouts
