# Horoscope Screen - Responsive Demo

## Overview
A simplified responsive demonstration of horoscope screen UI patterns.

## Features
- Zodiac sign grid selection
- Period tabs (Daily/Weekly/Monthly)
- Expandable prediction cards
- Date navigation
- Responsive layout patterns

## Zodiac Signs
All 12 zodiac signs with:
- Symbol (♈, ♉, etc.)
- Accent color
- Associated icon

## UI Components
- Tab bar for Daily/Weekly/Monthly
- Zodiac grid selector
- Date picker
- Expandable prediction cards
- Lucky elements display

## Prediction Categories
- Love
- Career
- Health
- Finance

## State
- Selected zodiac sign
- Selected tab (period)
- Selected date
- Expanded card states
