# Kundli Feature

## Overview
Generate and display detailed Vedic birth charts (Kundli/Kundali) with planetary positions, yogas, and doshas.

## Screens
| Screen | Route | Description |
|--------|-------|-------------|
| `kundli_input_screen.dart` | `/kundli/input` | Birth details form |
| `kundli_display_screen.dart` | `/kundli/display` | Kundli chart and analysis |

## Widgets
| Widget | Description |
|--------|-------------|
| `kundali_chart_painter.dart` | Custom chart drawing (North/South Indian) |

## Chart Styles
- **North Indian** - Diamond-shaped houses
- **South Indian** - Square grid layout

## Display Tabs
1. **Chart** - Visual birth chart
2. **Planets** - Planetary positions table
3. **Houses** - Bhava analysis
4. **Yogas** - Beneficial combinations
5. **Dasha** - Planetary periods

## Kundli Elements
- Lagna (Ascendant)
- Moon Sign (Rashi)
- Sun Sign
- Birth Nakshatra
- All 9 planets positions
- 12 house placements
- Yogas detection
- Dosha analysis (Manglik, etc.)
- Mahadasha/Antardasha periods

## Features
- Auto-generate from profile
- Save multiple kundalis
- Share as image/PDF
- Compare kundalis
- Detailed interpretations
