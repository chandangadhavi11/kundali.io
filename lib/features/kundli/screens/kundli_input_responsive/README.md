# Kundli Input Responsive

## Overview
Responsive demonstration of the Kundali birth details input form with adaptive layouts.

## Features
- Multi-step form wizard
- Responsive form layouts
- Location autocomplete
- Date/time pickers
- Gender selection
- Form validation

## Responsive Breakpoints
| Screen Size | Width |
|-------------|-------|
| Mobile | < 600px |
| Tablet | 600px - 900px |
| Desktop | > 1200px |

## Form Layouts
| Layout | Description |
|--------|-------------|
| Single | Mobile - single column |
| Two Column | Tablet - side by side |
| Multi Column | Desktop - multiple columns |

## Form Fields
| Field | Type | Required |
|-------|------|----------|
| Name | Text | Yes |
| Birth Date | Date Picker | Yes |
| Birth Time | Time Picker | Yes |
| Birth Place | Location | Yes |
| Gender | Selection | Yes |

## UI Components
- Stepper navigation
- Floating label text fields
- Date picker
- Time picker
- Location search with autocomplete
- Gender selection chips
- Progress indicator

## File
- `kundli_input_responsive.dart` - Responsive input widget

## Dependencies
- `intl` - Date formatting
- `app_colors.dart` - Theme colors
- `kundali_data_model.dart` - Data model
