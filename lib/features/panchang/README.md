# Panchang Feature

## Overview
Hindu calendar with daily panchang details, festivals, muhurat finder, and reminders.

## Screen
- `screens/panchang_screen.dart` - Main panchang view with tabs

## Widgets
| Widget | Description |
|--------|-------------|
| `month_calendar_view.dart` | Monthly calendar with tithis |
| `day_detail_view.dart` | Detailed daily panchang info |
| `festivals_index_view.dart` | Upcoming festivals list |
| `muhurat_finder_view.dart` | Auspicious timing finder |
| `reminders_view.dart` | Custom event reminders |

## Tabs
1. **Today** - Current day's panchang
2. **Calendar** - Monthly view
3. **Festivals** - Upcoming events
4. **Muhurat** - Auspicious times
5. **Reminders** - User notifications

## Panchang Elements
- Tithi (Lunar day)
- Nakshatra (Constellation)
- Yoga
- Karana
- Sunrise/Sunset times
- Moonrise/Moonset times
- Rahu Kaal, Gulika Kaal
- Abhijit Muhurat
