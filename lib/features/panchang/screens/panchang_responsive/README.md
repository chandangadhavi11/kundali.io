# Panchang Responsive

## Overview
Full responsive implementation of the Panchang screen with adaptive calendar, collapsible sections, and visualizations.

## Features
- Adaptive calendar layout
- Collapsible detail sections
- Sunrise/sunset visualization
- Tithi and Nakshatra display
- Muhurat timings
- Regional settings
- Responsive breakpoints

## Responsive Breakpoints
| Screen Size | Width |
|-------------|-------|
| Mobile | < 600px |
| Tablet | 600px - 900px |
| Desktop | 900px - 1200px |
| Ultra Wide | > 1800px |

## Panchang Elements
- **Tithi** - Lunar day
- **Nakshatra** - Lunar mansion
- **Yoga** - Lunisolar combination
- **Karana** - Half tithi
- **Var** - Day of week
- **Sunrise/Sunset** - Times

## UI Components
- Calendar grid/list view
- Date picker
- Detail cards
- Collapsible sections
- Time visualizations
- Regional selector

## File
- `panchang_responsive.dart` - Responsive panchang widget

## Dependencies
- `provider` - State management
- `intl` - Date formatting
- `app_colors.dart` - Theme colors
