# My Kundlis Screen

## Overview
Screen for managing saved Kundali charts for the user and family members.

## Features
- List of saved Kundalis
- Add new Kundali
- Edit existing Kundalis
- Delete Kundalis
- Primary Kundali indicator
- Animated card entrance

## Kundali Card Info
| Field | Description |
|-------|-------------|
| Name | Person's name |
| Relation | Relationship (Self, Spouse, etc.) |
| Birth Date | Date of birth |
| Birth Time | Time of birth |
| Birth Place | Place of birth |
| isPrimary | Primary Kundali flag |

## UI Components
- Animated card list
- Floating action button for adding
- Card with gradient accent
- Relation chip
- Primary badge
- Action buttons (Edit, Delete, View)

## Sample Relations
- Self
- Spouse

- Staggered card entrance
