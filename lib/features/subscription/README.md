# Subscription Feature

## Overview
Premium subscription plans and payment management.

## Screen
- `screens/subscription_screen.dart` - Subscription plans and purchase

## Plans

| Plan | Duration | Features |
|------|----------|----------|
| Free | - | Basic horoscope, limited kundlis |
| Monthly | 1 month | Full access, unlimited kundlis |
| Yearly | 12 months | Full access + priority support |
| Lifetime | Forever | All features, lifetime updates |
