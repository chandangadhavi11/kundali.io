# Subscription Screen

## Overview
Premium subscription management screen with plan comparison and purchase flow.

## Features
- Subscription plans display (Monthly/Yearly)
- Feature comparison table
- User testimonials carousel
- Current plan status
- Purchase/upgrade flow
- Animated transitions

## Subscription Plans
| Plan | Billing | Features |
|------|---------|----------|
| Free | - | Basic features |
| Monthly | Per month | Premium features |
| Yearly | Per year (discounted) | Premium + exclusive |

## Premium Features
- Unlimited Kundali generation
- Detailed predictions
- AI astrologer unlimited access
- PDF reports
- No ads
- Priority support
- Exclusive content

## UI Components
- Hero section with animated branding
- Plan selection cards
- Feature comparison table
- Testimonials carousel
- Subscribe/Upgrade button
- Current plan indicator

## Testimonials
Carousel of user reviews with:
- User name
- Rating
- Review text
- Auto-scroll functionality

## State Management
- Uses `AuthProvider` for user status
- Uses `ThemeProvider` for theming

## Animations
- Hero fade and scale
- Cards slide in
- Table expand
- Testimonial transitions

## State
- Selected plan index
- Current testimonial index
- Premium status
- Current plan details

## File
- `subscription_screen.dart` - Subscription management widget

## Dependencies
- `provider` - State management
- `font_awesome_flutter` - Icons
- `app_colors.dart` - Theme colors
